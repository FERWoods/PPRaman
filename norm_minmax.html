norm_minmax {PPRaman}R Documentation

Normalisation using min max

Description

Normalising function using min/max method in which the maximum intensity is set to 1 while other data points are scaled in comparatively.

Usage

norm_minmax(spectra)

Arguments

spectra

Spectra in matrix format for normalising

Value

Normalised spectra
