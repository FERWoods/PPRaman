Savitsky-Golay filtering SG filtering, adapted from pracma package

Description

Savitsky-Golay filtering SG filtering, adapted from pracma package

Usage

sav_gol(T, fl, forder = 4, dorder = 0)

Arguments

T

vector containing spectra, can be used with matrix of spectra using apply with MARGIN = 2

fl

Filter length - size of winder to smooth within

forder

Polynomial order for smoothing window

dorder

Derivative order, 0 for smoothing only, 1,2 = derivatives

Value

SG filtered spectra
