norm_snv {PPRaman}R Documentation

Normalising Spectra using SNV

Description

Normalises spectra using Standard Normal Variate method method which scales data based on the mean and standard deviation.

Usage

norm_snv(spectra)

Arguments

spectra

Spectral like data

Value

Normalised spectra using SNV method
