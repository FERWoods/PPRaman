modpolyfit {PPRaman}R Documentation

Modified polynomial baseline removal

Description

Function computing the modified polynomial baseline correction algorithm

Usage

modpolyfit(spectra, t, degree = 4, tol = 0.001, rep = 100)

Arguments

spectra

Matrix containing spectra

t

abcissa for spectra - optional

degree

polynomial degree for baseline fit

tol

tolerance of difference between iterations, default 0.001

rep

Max number of iterations

Value

Preprocessed spectra

References

CHAD A. LIEBER and ANITA MAHADEVAN-JANSEN: Automated Method for Subtraction of Fluorescence from Biological Raman Spectra
