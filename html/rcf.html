Rolling Circle Filter (RCF) Updated 12/08/2020

Description

Background removal for spectral data using RCF, Pchip used for baseline fit. RCF is a high pass filter which rolls under the spectra removing the broad baseline.

Usage

rcf(raw_spectra, radius, wavenumber)

Arguments

raw_spectra

Spectra data that has been interpolated already

radius

Numerical value for RCF radius selection

wavenumber

Wavenumber axis

Value

Corrected spectra with RCF applied (background removed)
